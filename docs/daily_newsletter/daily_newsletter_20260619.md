## 1. AI（人工知能）

### AIエージェントが自律的に自己改善する self-improvement スキルが良い
AIエージェントとの開発において、同じ説明を繰り返す手間を省くための「self-improvement」スキルの導入事例です。エージェントが会話の中で得た「学び」を構造化し、リポジトリ内の`.learnings/`ディレクトリにMarkdownファイルとして蓄積します。これにより、個人の経験がチーム全体の資産となり、AIがプロジェクトの文脈を学習しながら成長していく仕組みが構築できます [1]。

### AI Daily Digest 2026/6/18 — G7 AIサミット、オープンソースコーディングの躍進
G7サミットでのAIガバナンス協議や、中国Zhipu AIによる高性能オープンソースモデル「GLM-5.2」の公開など、グローバルなAI動向のまとめです。特に、MicrosoftがCopilotのコスト削減のためにオープンソースモデル（DeepSeek V4）の採用を検討しているというニュースは、エンタープライズAIの経済性に大きな影響を与える可能性があります [2]。

## 2. 心理学

### 自立を手放すときに怖れを感じるのは、とても自然なこと。
アドラー心理学の観点から、人が「自立」から「相互依存」のステージへ移行する際に感じる「怖れ」について解説しています。これまで一人で頑張ってきたやり方を手放すことへの抵抗感は自然なものであり、むしろプロセスが順調に進んでいる証拠であると捉える視点を提供しています。チーム開発やマネジメントにおいても、メンバーの心理的変化を理解する上で有用な洞察です [3]。

## 3. ビジネス

### 同じプロンプトを何度も書きたくないので、AIとのやりとりを棚卸しする仕組み作った
LayerX社における、AIへのプロンプトログを集計し、繰り返し行われている作業を自動化（Skill / Command / Schedule化）する取り組みの紹介です。AIへの指示をログとして記録し、LLMを用いて意味的にグルーピングすることで、頻出する作業パターンを抽出します。AIの使い方自体をデータに基づいて継続的に改善（リファクタリング）していくアプローチは、AIプロダクト開発の現場で非常に参考になります [4]。

## 4. IT（テクノロジー）

### Claude Code MCPサーバー接続ガイド（2026年6月）
Claude CodeからModel Context Protocol（MCP）サーバーに接続するための最新ガイドです。HTTPリモート、stdioローカル、WebSocketの3つのトランスポート方式や、Local / Project / Userの3つのスコープ設定について、具体的なコマンドと設定ファイル例を交えて解説しています。Tool Search機能によるコンテキスト節約や、セキュリティ上の注意点など、実践的な運用ノウハウがまとめられています [5]。

### AI需要の高まりでTSMCの生産能力が限界へ、代替パートナーとしてSamsungを検討中
AIブームによる半導体需要の急増により、TSMCの生産能力が限界に達しつつある現状を報じています。Google、NVIDIA、AMD、テスラなどの大手テクノロジー企業が、TSMCの代替としてSamsungのファウンドリ部門への製造委託を検討・拡大しており、半導体サプライチェーンの地殻変動が起きつつあります [6]。
