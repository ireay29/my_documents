# PetSense - 犬猫兼用スマート体調管理プラットフォーム

## 📌 プロジェクト概要

**PetSense** は、犬と猫の体重、排泄物（尿・便）の量と質を自動で計測・分析し、ペットの健康状態を可視化するスマートヘルスケアプラットフォームです。

### 核となる特徴

- **犬猫兼用**: 既存サービスが猫専用である中、犬にも対応
- **ボード型デバイス**: ペットシートの下に置くだけで、既存のトイレ環境に対応
- **高精度判定**: 感圧センサー + 振動センサーで尿・便を99%の精度で判定
- **自動給餌器連動**: フェーズ2で、体重・尿データに基づいた動的な給餌管理を実現
- **家族共有**: 複数のペット飼い主が同時にデータを管理・共有可能

## 🔧 技術スタック

### ハードウェア
- **マイコン**: nRF52840（Bluetooth LE 5.0）
- **センサー**: FSR 402（感圧）、BMI160（加速度計）
- **通信**: Bluetooth LE + Wi-Fi オプション
- **バッテリー**: 5000mAh（60日間連続稼働）

### アプリ
- **フロントエンド**: React Native（Expo）
- **バックエンド**: Node.js + Express（フェーズ2以降）
- **データベース**: SQLite（ローカル）→ PostgreSQL（フェーズ2以降）
- **言語**: TypeScript
