# PetSense デバイス要件定義書

**作成日**: 2026年3月25日  
**バージョン**: 1.0  
**対象**: PetSense ハードウェアプラットフォーム全体

---

## 1. 概要

PetSense は、犬猫のトイレに設置するスマートセンサーデバイスを中心に構成されます。本要件定義書は、以下の3つの主要デバイスと、それらを統合する IoT プラットフォームの技術仕様を定義します。

### 主要デバイス構成

| デバイス | 用途 | フェーズ | 優先度 |
| :--- | :--- | :--- | :--- |
| **スマートシーツセンサー** | 犬のトイレシーツ下に敷くセンサー。体重、尿量、排便を計測 | フェーズ1 | 最高 |
| **スマートトイレボード** | 猫の砂トイレ下に敷くボード。体重、尿・フン分析 | フェーズ1 | 最高 |
| **自動給餌器** | カロリー自動計算、体重・尿データに基づく給餌量調整 | フェーズ2 | 高 |
| **活動量計（首輪型）** | 運動量、心拍数、睡眠時間を記録 | フェーズ3 | 中 |
| **スマートウォーターボウル** | 飲水量、飲水パターンを記録 | フェーズ3 | 低 |

---

## 2. スマートシーツセンサー（犬用）

### 2.1 物理仕様

**サイズと形状**:
- 寸法: 400mm × 600mm × 8mm（標準的なトイレシーツサイズに対応）
- 重量: 150g（軽量で取り扱いやすい）
- 材質: 防水シリコン + 耐久性プラスチック
- 耐久性: 最低 1,000 回の排泄に耐える（約1ヶ月）

**防水性能**:
- IP67 等級（完全防水）
- 尿、フン、水に対する耐性
- 清掃時の水洗いに対応

### 2.2 センサー仕様

#### 2.2.1 圧力センサー（体重計測）

**仕様**:
- 計測範囲: 0.5kg～50kg（小型犬から大型犬まで対応）
- 精度: ±100g（0.1kg）
- 応答速度: 100ms
- 複数点圧力センサー配置（4点以上）で、不均等な体重分布にも対応

**機能**:
- リアルタイム体重計測
- 計測時間: 5秒以上の接触で記録
- 異常値フィルタリング（ノイズ除去）

#### 2.2.2 湿度・化学センサー（尿分析）

**仕様**:
- pH センサー: 範囲 5.0～9.0、精度 ±0.2
- 比重センサー: 1.005～1.030、精度 ±0.002
- 電気伝導度（EC）センサー: 0～2000 mS/cm
- タンパク質検出: 定性的（有無判定）
- ケトン体検出: 定性的（有無判定）

**機能**:
- 尿が接触した際に自動的に化学成分を分析
- 計測時間: 10～15秒
- 複数回計測による平均値の算出

#### 2.2.3 重量差分センサー（排便検知）

**仕様**:
- 排便前後の重量差を検知
- 最小検知重量: 5g（小型犬の排便に対応）
- 排便の硬さ推定: 重量変化の速度から判定

**機能**:
- 排便の有無を判定
- ブリストル・スケール（便の硬さ指標）を推定
- 異常な排便パターンを検知

### 2.3 通信仕様

**Bluetooth LE（BLE）**:
- バージョン: 5.0 以上
- 通信距離: 最低 10m（室内での使用を想定）
- 通信頻度: 1回の排泄ごとに 1回の送信
- データペイロード: 最大 100 bytes

**Wi-Fi**:
- オプション機能（将来の拡張用）
- 2.4GHz 帯域対応
- セキュリティ: WPA3

**データ形式**:
```json
{
  "device_id": "sensor_001",
  "timestamp": "2026-03-25T10:30:45Z",
  "event_type": "urination",
  "weight_kg": 3.2,
  "urine_ph": 6.8,
  "urine_specific_gravity": 1.020,
  "urine_protein": false,
  "urine_ketone": false,
  "duration_seconds": 12,
  "signal_strength": -45
}
```

### 2.4 電力管理

**バッテリー仕様**:
- 型式: リチウムイオン二次電池
- 容量: 5000mAh
- 電圧: 3.7V
- 連続稼働時間: 最低 30 日間（1日 5～10 回の排泄を想定）

**充電方式**:
- USB-C 充電
- 充電時間: 2～3 時間
- 充電サイクル: 最低 1,000 回

**電力消費**:
- スタンバイ時: 50μA
- 計測時: 500mA（15秒間）
- 通信時: 100mA（5秒間）

**低電力アラート**:
- バッテリー残量 20% で警告
- バッテリー残量 5% で自動シャットダウン

### 2.5 データ処理とローカルストレージ

**ローカルストレージ**:
- 容量: 最低 1MB（約 1,000 回の計測記録）
- 形式: SQLite
- 保存期間: 30 日間（その後、クラウドに同期）

**オンボード処理**:
- 異常値検出（外れ値除去）
- データ圧縮（ロスレス）
- タイムスタンプの自動付与

## 3. スマートトイレボード（猫用）

### 3.1 物理仕様

**サイズと形状**:
- 寸法: 450mm × 450mm × 10mm（標準的な猫砂トイレ下に敷く）
- 重量: 200g
- 材質: 防水シリコン + 耐久性プラスチック
- 耐久性: 最低 2,000 回の排泄に耐える（約2ヶ月）

**防水性能**:
- IP67 等級（完全防水）
- 猫砂、尿、水に対する耐性

### 3.2 センサー仕様

#### 3.2.1 圧力センサー（体重計測）

**仕様**:
- 計測範囲: 1kg～15kg（猫全般に対応）
- 精度: ±50g
- 応答速度: 100ms
- 複数点圧力センサー配置（6点以上）

#### 3.2.2 化学センサー（尿分析）

**仕様**:
- pH センサー: 範囲 5.0～9.0、精度 ±0.2
- 比重センサー: 1.005～1.030、精度 ±0.002
- 電気伝導度（EC）センサー: 0～2000 mS/cm
- タンパク質検出: 定性的
- ケトン体検出: 定性的
- 血液検出: 定性的（猫の泌尿器系疾患検知用）

#### 3.2.3 画像センサー（排便分析）

**仕様**:
- カメラ解像度: 最低 5MP（便の形状判定用）
- 撮影範囲: 90度視野角
- 赤外線 LED: 暗い環境での撮影に対応
- 画像保存: 最新 10 枚を保持

**機能**:
- 排便時に自動撮影
- AI による便の硬さ判定（ブリストル・スケール）
- 血便、寄生虫の兆候検知

### 3.3 通信仕様

**Bluetooth LE**:
- バージョン: 5.0 以上
- 通信距離: 最低 15m
- 通信頻度: 1回の排泄ごとに 1回の送信

**Wi-Fi**:
- 2.4GHz 帯域対応
- セキュリティ: WPA3

**データ形式**:
```json
{
  "device_id": "toilet_board_001",
  "timestamp": "2026-03-25T14:20:30Z",
  "event_type": "defecation",
  "weight_kg": 4.5,
  "urine_ph": 6.5,
  "urine_specific_gravity": 1.018,
  "urine_protein": false,
  "urine_blood": false,
  "feces_bristol_scale": 3,
  "feces_image_url": "s3://petsense-images/toilet_001_20260325_142030.jpg",
  "duration_seconds": 8
}
```

### 3.4 電力管理

**バッテリー仕様**:
- 容量: 8000mAh（猫は排泄回数が多いため）
- 連続稼働時間: 最低 45 日間
- 充電時間: 3～4 時間

### 3.5 データ処理とローカルストレージ

**ローカルストレージ**:
- 容量: 最低 4GB（画像保存用）
- 保存期間: 30 日間

## 4. 自動給餌器（フェーズ2）

### 4.1 物理仕様

**サイズと形状**:
- 寸法: 300mm × 300mm × 200mm
- 重量: 1.5kg
- 材質: 食品グレードプラスチック
- 容量: 3L（中型犬で 1 週間分）

### 4.2 機械仕様

**給餌メカニズム**:
- モーター駆動のスクリュー式
- 給餌精度: ±5g
- 最大給餌量: 500g/回
- 給餌速度: 50g/秒

**フード対応**:
- ドライフード: 粒径 5～15mm
- ウェットフード: 対応外（衛生上の理由）

### 4.3 センサー仕様

**重量センサー**:
- 計測範囲: 0～3000g
- 精度: ±10g
- リアルタイム監視

**温度・湿度センサー**:
- 温度範囲: 0～50℃
- 湿度範囲: 0～95%RH
- フード鮮度管理用

### 4.4 通信仕様

**Bluetooth LE**:
- バージョン: 5.0 以上
- 通信距離: 最低 20m

**Wi-Fi**:
- 2.4GHz 帯域対応
- クラウド連携用

**データ形式**:
```json
{
  "device_id": "feeder_001",
  "timestamp": "2026-03-25T08:00:00Z",
  "event_type": "feeding",
  "scheduled_amount_g": 150,
  "actual_amount_g": 148,
  "pet_id": "pet_001",
  "remaining_food_g": 2500,
  "food_freshness_hours": 24
}
```

### 4.5 スマート機能

**カロリー自動計算**:
- ペットの犬種、体重、年齢から適正カロリーを計算
- 体重・尿データに基づいて給餌量を動的に調整
- 1日 1～3 回の自動給餌

**スケジュール管理**:
- アプリから給餌時間を設定可能
- 複数ペットの個別スケジュール対応

## 5. 活動量計（首輪型、フェーズ3）

### 5.1 物理仕様

**サイズと形状**:
- 寸法: 40mm × 30mm × 15mm
- 重量: 15g（小型犬でも負担にならない）
- 材質: 防水シリコン
- 首輪への装着: 磁石式（簡単に取り外し可能）

### 5.2 センサー仕様

**加速度計**:
- 計測範囲: ±16g
- サンプリングレート: 100Hz
- 運動量計算: 加速度の積分値

**心拍センサー**:
- 計測範囲: 40～200 bpm
- 精度: ±5 bpm
- 計測間隔: 1 分ごと

**温度センサー**:
- 計測範囲: 35～42℃
- 精度: ±0.5℃
- 体温異常の検知

**GPS**:
- オプション機能
- 精度: ±5m
- 散歩ルートの記録

### 5.3 電力管理

**バッテリー仕様**:
- 容量: 200mAh
- 連続稼働時間: 7 日間
- 充電方式: ワイヤレス充電

### 7.1 クラウド連携

**データ同期**:

- 転送中: TLS 1.3
- 保存時: AES-256

**機能**:
- 複数のセンサーデバイスを集約
- ローカル処理による低遅延
- バックアップ電源（UPS）

**仕様**:
- 通信: Bluetooth LE、Wi-Fi、Ethernet
- ストレージ: 32GB SSD
- 消費電力: 5W
