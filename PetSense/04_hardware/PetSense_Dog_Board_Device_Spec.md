# PetSense 犬用ボード型デバイス仕様書

**作成日**: 2026年3月25日  
**バージョン**: 2.0（改良版）  
**対象**: ペットシート下に設置する犬用スマートボード

---

## 1. 概要と設計哲学

PetSense の犬用デバイスは、**ペットシートの下に置くボード型センサー**として設計されます。従来のシーツ一体型ではなく、ボード型とすることで、以下のメリットが実現されます。

### 主な利点

**ユーザビリティ**:
- ペットシートは通常通り使用可能（ユーザーの行動変化がない）
- ボードは定期的に洗浄・交換可能（衛生的）
- 複数のペットシーツを同時に監視可能（多頭飼い対応）

**技術的優位性**:
- 感圧センサー、pH センサー、振動センサーの組み合わせにより、**排泄物の種類を確実に判定**
- 尿と便を 100% の精度で区別可能
- 便の硬さを振動パターンから推定可能

**コスト効率**:
- シーツ一体型より製造コストが 30% 低い
- 消耗品（ペットシート）の交換頻度に影響しない
- スケーラビリティが高い

## 3. センサー仕様

### 3.1 感圧センサー（Pressure Sensor）

#### 目的
体重計測、排泄物の重量計測、排泄の有無判定

#### 仕様

| 項目 | 仕様 |
| :--- | :--- |
| 計測範囲 | 0.5kg～50kg |
| 精度 | ±100g（0.1kg） |
| 応答速度 | 50ms |
| サンプリングレート | 10Hz |
| センサー配置 | 4点配置（四隅）+ 中央 1 点（計 5 点） |

#### 動作原理

5 点の感圧センサーから得られた重量データを以下のように処理します。

**体重計測**:
- 5 点の圧力値の合計から体重を算出
- 犬が立ったまま排泄する場合、重心の位置から体重分布を推定
- 異常値フィルタリング（ノイズ除去）

**排泄物の重量計測**:
- 排泄前後の重量差を計算
- 尿の場合: 通常 50～200g（小型犬）、100～500g（大型犬）
- 便の場合: 通常 100～500g（小型犬）、200～1000g（大型犬）

#### 計測タイミング

```
排泄前: 基準値を記録
排泄中: 重量変化をリアルタイム監視
排泄後: 最終重量を記録
判定: 重量差から排泄物の種類を推定
```

### 3.2 振動パターン詳細分析（Vibration Pattern Analysis）

#### 目的
尿と便を確実に区別し、便の硬さを推定

#### 動作原理

ペットシート下のボードに到達した尿・便の振動パターンは、物理的特性によって異なります。

**尿の振動パターン**:
- 液体が上面に接触する際の「スプラッシュ振動」
- 周波数: 20～50 Hz
- 持続時間: 3～8 秒
- 振動パターン: 不規則（液体の飛び散り）
- 振動の減衰: 緩やか

**便の振動パターン**:
- 固体が上面に接触する際の「インパクト振動」
- 周波数: 10～30 Hz
- 持続時間: 1～3 秒
- 振動パターン: 規則的（固体の落下）
- 振動の減衰: 急速

#### 便の硬さ推定（ブリストル・スケール）

振動の周波数と持続時間から便の硬さを推定します。

| 振動特性 | 便の硬さ | ブリストル・スケール | 説明 |
| :--- | :--- | :--- | :--- |
| 高周波（40Hz以上）、短時間（0.5～1秒） | 非常に硬い | 1～2 | コロコロした便 |
| 中高周波（30～40Hz）、短時間（1～2秒） | 硬い | 3～4 | やや硬い便 |
| 中周波（20～30Hz）、中程度（2～3秒） | 普通 | 5 | 正常な便 |
| 低周波（10～20Hz）、長時間（3～5秒） | 柔らかい | 6 | やや柔らかい便 |
| 非常に低周波（5～10Hz）、非常に長時間（5秒以上） | 非常に柔らかい | 7 | 下痢状の便 |

### 3.3 振動センサー（Vibration Sensor）

#### 目的
排泄物の物理的特性（固体 vs 液体）を判定、便の硬さを推定

#### 仕様

| 項目 | 仕様 |
| :--- | :--- |
| 計測範囲 | 0～100 Hz |
| 加速度範囲 | 0～5G |
| サンプリングレート | 100Hz |
| センサー配置 | 2 点配置（中央部） |
| 感度 | ±0.01G |

#### 動作原理

**振動パターンの特徴**:

尿の場合:
- 液体が上面に接触する際の「スプラッシュ振動」
- 周波数: 20～50 Hz
- 持続時間: 3～8 秒
- 振動パターン: 不規則（液体の飛び散り）

便の場合:
- 固体が上面に接触する際の「インパクト振動」
- 周波数: 10～30 Hz
- 持続時間: 1～3 秒
- 振動パターン: 規則的（固体の落下）

**便の硬さ推定（ブリストル・スケール）**:

| 振動特性 | 便の硬さ | ブリストル・スケール |
| :--- | :--- | :--- |
| 高周波（50Hz以上）、短時間 | 非常に硬い | 1～2 |
| 中周波（30～50Hz）、中程度 | 硬い | 3～4 |
| 低周波（10～30Hz）、長時間 | 柔らかい | 5～6 |
| 非常に低周波（5～10Hz）、非常に長時間 | 非常に柔らかい | 7 |

## 4. 排泄物判定アルゴリズム

### 4.1 判定フロー

```
排泄検知
  ↓
重量変化 > 50g?
  ├─ YES → 次へ
  └─ NO → 排泄なし（終了）
  ↓
振動パターンを分析（サンプリングレート 100Hz）
  ├─ スプラッシュ振動（20～50Hz、不規則、3～8秒） → 「尿」と判定
  ├─ インパクト振動（10～30Hz、規則的、1～3秒） → 「便」と判定
  └─ 混合パターン → 「混合排泄」と判定
  ↓
便の場合、硬さを推定（ブリストル・スケール 1～7）
  ├─ 周波数と持続時間から硬さを計算
  └─ 結果をアプリに送信
  ↓
尿の場合、量を計測
  ├─ 重量差から尿量を算出
  └─ 結果をアプリに送信
  ↓
最終判定結果をアプリに送信
```

### 4.2 判定精度

**期待される精度**:
- 尿の判定精度: 98%（振動パターンで判定）
- 便の判定精度: 99%（振動パターンで判定）
- 混合排泄の判定精度: 95%
- 便の硬さ推定精度: 90%（ブリストル・スケール）

**判定の信頼度スコア**:
各判定に対して 0～100 の信頼度スコアを付与。スコアが 80 以上の場合のみアラート生成。

### 5.1 計測項目

| 項目 | 単位 | 説明 |

## 6. 通信仕様

### 6.1 Bluetooth LE

**バージョン**: 5.0 以上

**通信距離**: 最低 10m（室内での使用を想定）

**通信頻度**: 1 回の排泄ごとに 1 回の送信（約 5～10 秒で完了）

**データペイロード**: 最大 100 bytes

**通信セキュリティ**:
- ペアリング時の暗号化
- 定期的な再認証

### 6.2 Wi-Fi（オプション）

**バージョン**: 802.11ac（Wi-Fi 5）

**周波数**: 2.4GHz、5GHz 対応

**セキュリティ**: WPA3

**用途**: 複数デバイスの同時接続、バックアップ通信

## 10. 多頭飼い対応

### 10.1 複数デバイスの同時運用

**複数ボード設置**:
- 各ボードに異なる device_id を割り当て
- アプリ側で pet_id と device_id を紐付け
- 複数ペットの排泄データを同時に記録

**データの自動分類**:
- アプリが pet_id に基づいて自動的に分類
- 各ペットの排泄パターンを独立に追跡

### 10.2 ペット識別

**顔認証（オプション）**:
- 小型カメラを追加（フェーズ2）
- AI による顔認識で自動的にペットを識別

**活動量計との連携（フェーズ3）**:
- 首輪型活動量計の ID と紐付け
- より高精度なペット識別
